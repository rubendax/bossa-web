BOSSA Web Firmware Flasher

Easy firmware flashing for ATSAMD21-based devices

How to Use

  1. Select your firmware file (.bin from Arduino IDE or PlatformIO)
  2. Click "Flash Firmware" - The tool will:
    • Ask you to select your device's serial port
    • Automatically reset your device into bootloader mode
    • Flash the firmware
    • Reset the device to run the new code

Supported devices:
ATSAMD21-based boards (Adafruit Feather M0, Arduino Zero, MKR series)

Supported browsers:
Chromium-based browsers (Brave 1.22+, Chrome 89+, Edge 89+, Opera 76+, Vivaldi 3.7+)
